    PROJECTS

    Project # 1

    • Client:Montgomery
    • Software:Micro station
    • Role:data capturing, Conversion
    • Team Size: 15

    Description:

    This project aims at migration of 3D planimetric data from design file to geo-database.
    Initially the design file data consisting of transportation features, hydrological features,
    building features and vegetation data was updated in 3D using photogrammetry software.
    The updated features were divided into blocks for convenience and were converted to individual geo-database files.
    All the topological errors such as dangles; label errors were checked in AutoCad.
    Then attributes were linked so as to differentiate the old and updated data. Finally, the data was sent to the client in geo-database format
